# Frontend (React + Vite)

## Install
```bash
npm install
```

## Run (dev)
```bash
npm run dev
```

## Build
```bash
npm run build
```

## Environment
Copy `.env.example` to `.env` when needed.

- `VITE_API_URL` - API base URL (for prod/Pages)
- `VITE_BASE_PATH` - Vite base path (used by Pages workflow)
